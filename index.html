Perguntas

A) Na tag "input" com atributo type = "text", para que serve o atributo "placeholder" ?

O atributo "placeholder" em uma tag "input" com o atributo type= "text" é usado para fornecer um texto de exemplo ou uma dica sobre o que deve ser inserido no campo de entrada. Ele é exibido dentro do campo antes que o usuário insira qualquer texto e desaparece assim que o campo recebe o foco.

B) Na tag "input" com atributo type = "radio" ou type = "radio", para que serve o atributo "checked" ?

O atributo "checked" em uma tag "input" com o atributo type="radio" ou type="checkbox" é usado para indicar se o elemento está selecionado ou marcado por padrão.

C) Na tag "input" para que serve o atributo "value" ?

O atributo "value" em uma tag "input" é usado para definir o valor padrão ou o valor atual do campo de entrada. Ele especifica o valor que será enviado ao servidor quando o formulário for enviado ou que será acessado por meio de scripts no lado do cliente.

D) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez ?

Para agrupar vários botões de rádio em um formulário e garantir que apenas um deles fique marcado de cada vez, é necessário utilizar o atributo "nome" com o mesmo valor para todos os botões de rádio do grupo.

E) Para que serve a tag "label" ? Para que serve o atributo "for" nessa tag ?

A tag "label" é usada para associar um rótulo descritivo a um elemento de entrada em um formulário HTML. Ela é usada para fornecer uma descrição textual ou um rótulo visível para um elemento de entrada, como um campo de texto, botão de opção ou caixa de seleção.

F) Na tag "option" para que serve o atributo "selected" ?

O atributo "selected" na tag "option" é usado para especificar qual opção em um elemento "select" deve ser exibido como selecionado por padrão quando a página é carregada. Ele indica qual opção deve ser pré-selecionada quando o usuário visualiza o menu suspenso.

G) Nas tags de formulário para que serve o atributo "required" ?

O atributo "required" em tags de formulário é usado para especificar que um campo de entrada deve ser preenchido antes que o formulário possa ser subordinado. Ele indica que o campo é obrigatório e não pode ser deixado em branco pelos usuários.

H) Na tag "form" para que serve o atributo "action" ?

O atributo "action" na tag "form" é usado para especificar onde os dados do formulário devem ser enviados quando o formulário é submetido

I) Na tag "form" para que serve o atributo "method" ? Quais são os valores possíveis desse atributo ?

O atributo "method" na tag "form" é usado para especificar o método HTTP a ser usado ao enviar os dados do formulário para o servidor.a

J) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form" ?

O valor do atributo "method" na tag "form" não afeta diretamente a visibilidade do valor digitado em um campo de senha na barra de endereços do navegador. O atributo "método" é usado para especificar o método HTTP utilizado para enviar os dados do formulário para o servidor.

K) Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos ?

No caso de um formulário que será usado para fazer upload de arquivos, o valor do atributo "method" na tag "form" deve ser definido como "POST". O método "POST" é comumente usado para envio de dados, incluindo arquivos, para o servidor
